函数类型：print函数（打印）input（结果是字符串）int（将字符串转换成整数）str（将整数转换成字符串）range(0,10,2)从零到十，每个间隔2
a = input('这是提示内容')  

if a == 8:
    print(9)
elif a == 9:
    print(7)
else:
    print(5)

while a < 9:
    print('*')
    a = a + 2
    
while 1:
    print('死循环')
while True:
    print('死循环')
    
for i in range(10):
    print(i)
